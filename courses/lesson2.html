Lesson 2: Variables and Data Types

📘 What You’ll Learn

In Python, variables are used to store data. You don’t have to declare a type — Python figures it out!

  • int – Whole numbers like 5
  • float – Decimal numbers like 3.14
  • str – Text strings like "Hello"
  • bool – Boolean values: True or False
