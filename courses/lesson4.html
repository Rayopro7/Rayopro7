Lesson 4: Conditional Statements

🧩 What You’ll Learn

Conditional statements allow you to make decisions in your Python code. They help your program behave differently based on conditions.

  • if – Executes a block of code if the condition is true
  • elif – Executes if the previous condition was false and this one is true
  • else – Executes if none of the above conditions were true

Example:

age = 18
if age >= 18:
    print("You're an adult")
else:
    print("You're a minor")
